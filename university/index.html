Format

Every lesson follows the same shape.

1. What you'll learn

The outcome in one sentence. If that's already clear to you, skip the lesson.

2. Why it matters

The problem this solves — so when you hit the problem, you remember the fix.

3. Walkthrough

Real terminal output, real screenshots, real copy-pastable commands. No pseudocode.

4. Gotchas

The mistakes we saw beta users make. Read these before you spend 20 minutes debugging.
