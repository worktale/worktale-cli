Commands

worktale

Alias for worktale dash. Opens the TUI dashboard.

worktale init

Import history of the current git repo, register it, install post-commit hook.

worktale dash

Open the interactive TUI (Overview / Daily Log / History / AI Sessions).

worktale today

Quick summary of today (commits, lines, files, coding time).

worktale status [--all-repos]

One-line status. --all-repos aggregates every tracked repo.

worktale log [--days N] [--since DATE] [--until DATE] [-f json|csv|markdown]

Browse daily activity log. Format flags emit structured output.

worktale digest [--date YYYY-MM-DD] [-f json|markdown]

Generate a digest for today (default) or a specific date.

worktale note "<text>"

Append a note to today's user_notes. Shows up in digest + Daily Log.

worktale publish [--date YYYY-MM-DD]

Publish a saved digest to Worktale Cloud. Requires login.

worktale standup [--days N] [--format slack|markdown|email|html] [--all-repos]

Generate an AI-written recap post.

worktale repos [list | rename <name> <new> | move <name> <path> | remove <name>]

Manage tracked repos.

worktale batch <dir> [--since N[mdy]] [--auto] [--annotate] [--install-hooks] [--exclude <regex>]

Recursively find and import every git repo under <dir>.

worktale hook <install|uninstall|status> [<repo-path>]

Manage the post-commit hook in a repo.

worktale nudge <--install|--uninstall|--check>

Shell-profile reminder at end of day.

worktale session <add|list|stats> [flags]

AI session management. add inserts a session row; pass --write-note to also append the --note text to today's daily log in one call (v1.5.0+). list and stats support --format.

worktale config <get|set|unset> [key] [value]

Read/write config values.

worktale cloud <login|logout|status|profile>

Cloud login and identity.

worktale export [--format json|csv] [--dir <path>]

Dump the full database to JSON or CSV files.

worktale diagnose

Print version, OS, config (redacted), DB info. Paste into bug reports.

worktale capture <repo-path>

Internal: called by git post-commit hook. You don't run this directly.

worktale --version

Print CLI version.

worktale --help [command]

Show help; pass a command for command-specific help.

Global flags

FLAGMEANING
-f, --formatOutput format (log/digest/session): json, csv, markdown
--days NLookback window in days
--since <d>Start date: ISO or Ndays/Nweeks/Nmonths (e.g. 3m)
--until <d>End date (inclusive)
--date <d>Specific date for single-day commands
--all-reposAggregate across every tracked repo
-h, --helpShow help and exit
-v, --versionPrint version and exit

Database schema

Every row is local. Nothing here is transmitted unless you explicitly publish.

TABLEPURPOSE
reposEvery tracked repo: path, name, first_seen, last_synced
commitsOne row per commit: sha, msg, author, timestamp, stats, branch, tags
daily_summariesOne per (repo, date): counts, notes, ai_draft, published flag
file_activityPer (repo, path, date): change counts for module aggregation
ai_sessionsOne per AI session: provider, model, tool, cost, tokens, tools_used
