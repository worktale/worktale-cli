Local first. Cloud when you’re ready.

01

Install the CLI

npm install -g worktale — then run worktale init in your repos. Your entire git history is captured locally.

02

Connect to Cloud

Sign up at worktale.dev, then run worktale cloud login. One-time setup.

03

Publish

Run worktale publish and your record goes live at worktale.dev/yourname.
